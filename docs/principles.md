# Principles

1. Keep the platform simple.
2. Separate platform from personal runtime.
3. Reuse proven frameworks.
4. Evolve modules independently.
5. Version the platform like software.
6. Optimize continuous improvement over perfect planning.
7. Treat personal management as an engineering discipline.

## Operating Heuristics

- Use small experiments for process changes.
- Review metrics before redesigning workflows.
- Limit work in progress to preserve flow.
- Prefer explicit definitions over hidden assumptions.
