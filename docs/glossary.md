# Glossary

- Vision: Long-term direction.
- Goal: Measurable desired outcome.
- Program: Strategic container that delivers goals.
- Operating Model: Repeatable way a program is operated.
- Management Kernel: Reusable management capability layer.
- Execution: Actual work performed in external systems.
- Review: Cadence-based reflection and improvement loop.
- WIP: Work in progress.
- ADR: Architecture/Decision Record.
- SemVer: Semantic Versioning (MAJOR.MINOR.PATCH).
