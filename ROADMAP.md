# Roadmap

## Guiding Direction

Build a modular, tool-agnostic management platform that can support multiple personal runtime repositories.

## Near Term (v1.x)

- Stabilize kernel module interfaces (inputs/outputs/metrics).
- Improve examples in module templates and playbooks.
- Add module maturity levels and adoption guidance.
- Define lightweight governance for platform changes.

## Mid Term (v2.x candidate)

- Introduce cross-module orchestration patterns.
- Define interoperability contract for external tools (Jira, Calendar, Notes).
- Add reference implementation examples without personal data.

## Future Scope (Not v1)

- Operating Model Generator.
- AI Planning Engine.
- Calendar Scheduler.
- Resource Optimization.
- Recommendation Engine.
- Personal Analytics Dashboard.
